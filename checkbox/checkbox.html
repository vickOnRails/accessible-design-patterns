Accessible Toggle Modal

The toggle works by setting aria-selected to true or false based on if the checkbox is checked or not. There's also a pseudo element for the mobile checkbox ball that is transitioned based on the checkbox value

I've tested for accessibility with NVDA, Windows Narrator and Talkback. Please reach out to me if you discovered something's wrong. You can find the code for this, alongside other of my accessibility experiments on github
